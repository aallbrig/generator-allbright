#### Generator Allbright

Andrew Allbright specific "meta-programming" code generators.

#### Installation
From [NPM](https://www.npmjs.com/package/generator-allbright)
``` bash
npm install yo -g
npm i generator-allbright -g
```

Build from source
``` bash
npm link
```

##### Usage
``` bash
yo allbright
yo allbright:gitignore
yo allbright:bash-basic-script
```
